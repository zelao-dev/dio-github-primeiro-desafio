Introdução ao Git e Github

Anotações Gerais

  • iniciar repositório local (é necessário depois enviar para o Github):

    git init

  • Adicionar usuário e e-mail (de preferência, nome e e-mail devem ser iguais à conta do Github):

    git config --global user.name nomedeusuario

    git config --global user.email "exemplo@email.com"

  • Adicionar todos os arquivos à STAGE:

    git add .

  • Adicionar arquivo específico à STAGE:

    git add arquivo-de-exemplo.txt

Desafio

  1. Iniciar um repositório remoto Público por meio do Github, utilizando minha conta pessoal
  2. Na minha máquina (repositório local) clonar o repositório. Comando:

    git clone https://linkdorepositorio

  3. Nomear o arquivo anotacoes.txt para anotacoes.html (pode ser usado o comando):

    mv anotacoes.txt anotacoes.html

  4. Fazer um commit no repositório local:

    git commit -m "mensagem do commit"

  5. Empurrar a situação do repositório local para repositório remoto (push):

    git push origin main
